# Multilingual Patient Management System

This project is a web-based application designed to manage patient records with multilingual support.
It allows users to store patient details, manage patient history, and handle translations in multiple languages.

## Features
•⁠  ⁠Add and manage patient information
•⁠  ⁠Multilingual support for patient details
•⁠  ⁠Patient history tracking
•⁠  ⁠Simple and user-friendly interface

## Technologies Used
•⁠  ⁠HTML
•⁠  ⁠JavaScript
•⁠  ⁠Python
•⁠  ⁠VS Code

## Purpose
This project was developed as an academic learning project to apply programming, data handling in a real world inspired scenarios.
